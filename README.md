Simple Phaser 3 starter demonstrating A and S movement.

How to run

1. Install dependencies:

```
npm install
```

2. Start local server:

```
npm start
```

3. Open http://localhost:8080 in your browser.

Controls

- `A`: move left
- `S`: move right
